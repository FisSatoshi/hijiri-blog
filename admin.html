管理画面

新しいタグを入力すると、記事一覧に自動で絞り込み項目が増えます。

この管理画面は GitHub の個人アクセストークン（PAT）を使って、直接あなたのリポジトリにデータを保存します。 トークンはこの端末の localStorage にのみ保存され、他の場所には送信されません。

トークンの作り方：GitHubの Settings → Developer settings → Personal access tokens → Fine-grained tokens で、 このリポジトリだけに対象を絞り、「Contents」の権限を Read and write にして発行してください。
